 LlanquihueTour

 Descripción

LlanquihueTour es un proyecto desarrollado en Java que simula el funcionamiento. El sistema permite registrar clientes, guías, proveedores, tours y reservas utilizando Programación Orientada a Objetos.

 Funcionalidades

- Registrar clientes.
- Registrar guías.
- Registrar proveedores.
- Registrar tours.
- Registrar reservas.
- Buscar clientes por RUT.
- Filtrar tours por destino.
- Validar RUT mediante una excepción personalizada.
- Guardar información en archivos de texto.

Clases principales

- Persona.

- Guia.
- Proveedor.

- Reserva.
- SistemaTurismo.
